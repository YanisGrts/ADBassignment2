Voici une liste de database que j'ai trouvé qui pourraient être sympa pour le notebook

## BioMedical
bdd dans le domaine biomédical qui a été développée en 2021 pour pouvoir faire des recherches rapide dans le domaine de la santé
Ex: QUels traitements a montré la meilleur efficacité sur certains symptomes. 
https://www.kaggle.com/datasets/rumjotkaur/pico-based-biomedical-knowledge-graph-neo4j-dump
Article de recherche des gens ayant développé ça : https://ceur-ws.org/Vol-3004/paper8.pdf


## Football
bdd regroupant tout les résultats de matchs de league anglaise depuis 1997. 
https://www.football-data.co.uk/englandm.php

## Airport Route 
A global airport route database encompassing airports worldwide, offering opportunities to apply graph algorithms and investigate spatial geolocalization.
https://github.com/neo4j-graph-examples/airport-routes


## Film recommandation:
